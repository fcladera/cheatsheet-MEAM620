\subsection*{Plane constraint}
Eg: $A X_w + B Y_w + C Z_w = 1$
\begin{itemize}
  \item Substitute the 1 in the last position. Expand.
  \item Replace $Z_w = 1/C (A X_w + B Y_w)$
\end{itemize}
